Project overview

The Kai&Nova repo hosts the entire autonomous architecture: `src/kai-nova.js` runs the agent loop, `scripts/` deploys to OpenClaw, `powers/` expands tools, and `.jork/` keeps the twins' memory, ledger, and goals.

  • Node.js 18+ stack with dotenv, OpenAI/Anthropic drivers
  • Telegram bot orchestrated through Hermes (pm2 optional)
  • LLM provider agnostic: Claude CLI, OpenAI, Xiaomi MiMo, Anthropic, or any OpenAI-compatible API

Automation scripts

`scripts/deploy-openclaw.sh` pushes commits through `hermes openclaw run` (gateway token + mission name envs). `scripts/hermes-watch-repo.sh` watches key folders and reruns deployments automatically.

OpenClaw + Hermes

Use `hermes openclaw board`/`hermes openclaw run` to coordinate tasks. OmniRoute keeps LLM traffic secure. The twin telemetry logs to `workspace/.jork/jork.log` and `pm2` for visibility.

Trail · Deploy

Every deployment records mission name, commit hash, timestamp, and Hermes notice in the Power log for traceability.

Trail · Goals

Goals + steps live in `.jork/goals.json`. Kai&Nova pick pending steps, mark progress, and update `SNAPSHOT.md` each cycle.

Trail · Ledger

`LEDGER.md` tracks rewards so the twins know what value was captured and what to pursue next.

Powers catalog

`powers/INDEX.md` enumerates each tool: web-search, web-fetch, memory-search, observability, and any future helpers you add. The twins load README instructions and scripts on demand.

Observability power

`powers/observability/collect.sh` pulls the last 200 lines from `jork.log`, appends `pm2 status`, and writes it to `workspace/powers/observability/output.txt` for the next cycle.

Power · web-search

Runs DuckDuckGo queries via bash, summarizes results, and feeds them back into the agent context.

Power · web-fetch

Fetches target URLs and produces summaries you can cite in Kai&Nova’s journal or Telegram updates.

Power · memory-search

Searches `.jork/history.jsonl` and `JOURNAL.md` to recall lessons, then stores results in `workspace/powers/memory-search/` for referential continuity.
