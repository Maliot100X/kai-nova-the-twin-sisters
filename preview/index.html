Core Demo Highlights

Deploy this folder to Vercel as a static site and share the Kai&Nova story with a glowing UI.

Live telemetry

Data streams from `workspace/.jork` plus `jork.log` and Hermes watchers keep every action traceable.

Automation suite

Scripts `scripts/deploy-openclaw.sh` and `scripts/hermes-watch-repo.sh` push commits into OpenClaw automatically.

Powers & insights

Built-in `powers/observability`, `web-search`, `web-fetch`, `memory-search` illustrate how Kai&Nova extend themselves.

Multi-LLM ready

Switch between Claude CLI, Anthropic, OpenAI, Xiaomi MiMo or other OpenAI-like endpoints by editing `.env`.

Deploy the preview

  1. Point Vercel's root directory to preview/ when importing the repo.
  2. Set an environment variable `NEXT_PUBLIC_KAI_NOVA_TAG` (optional) to customize the hero tagline.
  3. The static HTML + CSS run on Vercel without build steps.

Example Vercel CLI command: vercel --prod --root preview/ --name kai-nova-demo

4
