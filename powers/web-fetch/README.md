# web-fetch

Type: bash

Purpose: Fetch a URL and hand Kai&Nova the raw contents plus a short explanation of what was found.

Usage:
1. Prepare a text file at `workspace/powers/web-fetch/target.txt` with the URL.
2. Run the helper script to download the body (curl + sed) and summarize the key findings.
3. Store the results in a note Kai&Nova can read during their think cycle.
