# memory-search

Type: bash

Purpose: Search Kai&Nova's journal, SNAPSHOT, and history for prior conversations, goal states, or lessons.

Usage:
1. Drop a keyword list into `workspace/powers/memory-search/query.txt`.
2. The power scans `.jork` files (journal, snapshot) for matches and returns lines with context.
3. Kai&Nova cite the findings back in their conversation when they need continuity.
